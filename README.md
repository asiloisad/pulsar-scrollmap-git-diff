# scrollmap-git-diff

Git diff markers for [scrollmap](https://github.com/asiloisad/pulsar-scrollmap). Shows added, modified, and removed lines.

## Installation

To install `scrollmap-git-diff` search for [scrollmap-git-diff](https://web.pulsar-edit.dev/packages/scrollmap-git-diff) in the Install pane of the Pulsar settings or run `ppm install scrollmap-git-diff`. Alternatively, you can run `ppm install asiloisad/pulsar-scrollmap-git-diff` to install a package directly from the GitHub repository.
